## 转座元件

判断一个检测到的 **转座元件（TE, transposable element）是否完整**，通常需要结合 **序列长度、结构特征、边界完整性和功能元件** 等信息。下面我给你详细说明判断方法和实践策略：



## **1. 基于序列长度**

* **参考 TE 库**（Dfam、Repbase）提供了各类 TE 的完整长度范围。
* **方法**：

  1. 比对检测到的 TE 序列与 consensus 序列
  2. 计算覆盖率：`coverage = 检测 TE 长度 / consensus TE 长度`
  3. 常用阈值：覆盖率 ≥ 90% 可认为是 **完整 TE**；覆盖率较低可能是截断或缺失
* **工具**：

  * TLDR、TELR 输出的 TE assembly 通常给出长度和覆盖率
  * RepeatMasker 可标注 TE 的边界，并判断是否完整



## **2. 结构特征检查**

不同 TE 类型有特定结构：

| TE 类型                   | 完整特征                                                    |
| ----------------------- | ------------------------------------------------------- |
| **LINE1 (L1)**          | 5’UTR + ORF1 + ORF2 + 3’UTR + polyA tail；缺失 5’ 端常见于断裂插入 |
| **SINE (Alu)**          | 两个类似的 monomer + A-rich 3’端；缺失任何 monomer 可判断为不完整         |
| **LTR retrotransposon** | 左右长末端重复 (LTRs) + 内部 gag/pol；缺失 LTR 或内部序列为不完整            |
| **DNA transposon**      | TIR（末端倒位重复）完整，编码转座酶 ORF 是否完整                            |

**判断方法**：

* 使用比对工具（BLAST、MAFFT、minimap2）比对 TE 序列与参考库
* 检查关键结构元素是否存在



## **3. 边界完整性**

* **端点完整**：完整 TE 的 5’ 和 3’ 边界与参考 TE 对齐
* **缺失情况**：

  * 长读测序中常见 5’ 端截短 → 不完整
  * 内部缺失 → 结构 TE fragment

**实践方法**：

* 检查 VCF / assembly 输出中的 TE 边界坐标
* 如果 TE 端点对齐参考序列端点 → 完整



## **4. 功能元件检查**

* 完整 TE 通常包含：

  * ORF 或转座酶编码区（LINE, LTR）
  * 调控序列（启动子、polyA tail, 内部 promoters）
* 可以用 ORF finder 或 HMM 模型检测是否完整



## **5. 综合判断标准**

1. **长度覆盖率 ≥ 90%**
2. **结构完整**（关键域、LTR、TIR、ORF）
3. **边界完整**（5’ 和 3’ 对齐参考序列）
4. **功能元素存在**（可选，根据研究目标）

> 满足以上条件的 TE 可以认为是 **完整 TE**。如果只有部分覆盖、缺少关键结构或端点截断，则记为 **不完整 / fragment**。



## **6. 常用工具与策略**

| 工具                           | 功能                            |
| ---------------------------- | ----------------------------- |
| **RepeatMasker**             | 标注 TE 及边界，提供覆盖长度信息            |
| **TELR / TLDR**              | 长读 TE 检测，输出 assembly 长度、完整性评估 |
| **Dfam / Repbase**           | 提供参考 TE 序列和结构信息               |
| **MAFFT / minimap2 / BLAST** | 检查 TE 与参考比对完整性                |
| **ORFfinder / HMMER**        | 检查 TE 编码区 ORF 完整性             |



💡 **实践建议**：

1. 用长读测序得到 TE 序列 → 汇总 SV / insertion → 提取 ALT 或 assembly
2. 与 TE consensus 比对 → 计算覆盖率 → 初步筛选完整 TE
3. 检查结构元素（LTR、ORF1/2、TIR） → 进一步判定
4. 根据边界位置、功能域 → 最终判定完整性
